## Initial Roadmap for Conclave Development

### Phase 0
- [X] Basic encrypted networking.
- [X] Unit tests showing basic functionality works.
- [ ] Create a basic Client, Server, Tracker.
- [ ] Create a basic Client GUI.
- [ ] Client can connect to Tracker, list Servers, and connect to Server.
- [ ] Client can see other users, and see when a user joins or leaves.

### Phase 1
- [ ] Add chatroom functionality.
- [ ] Add direct message functionality.
- [ ] Add server administration functionality.
- [ ] Show users: status, group memberships (roles), administrator status.

### Phase 2
- [ ] Add forums.
- [ ] Add file sharing and permissions for shared files and filesystem.
- [ ] Add optional chat and forum content expiration.

### Phase 3
- [ ] Add rich text support, possibly including graphics.

## Possible Future Features
- [ ] Web-based client, possibly using WebAssembly.
- [ ] Mobile client.
- [ ] Audio/Video sharing. Streaming. Video chat.
- [ ] End-to-end encryption.
- [ ] Plugin systems for servers, clients; possibly using WebAssembly.
- [ ] Python module for the client, possibly useful for bots or automation.
